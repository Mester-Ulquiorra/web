# UCP frontend

The frontend of the Ulquiorra UCP (User Control Panel). This does not contain the backend API!

## Build instructions

1. Clone the repository

   ```sh
   git clone https://github.com/Mester-Ulquiorra/web.git
   cd web
   ```

2. Start the development server.

   ```sh
   bun install
   bun dev
   ```

   This will start a development server that you can view by opening `http://localhost` in your browser.
